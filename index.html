CYOA

Our first big project of the year was our choose your own adventure webpage. This was designed to teach us how to format a webpage, get familiar with displaying text, and use links to progress through a webpage.

As an additional challenge, I decided to add a tracker to keep you progress saved and a map to display the paths you have already discovered so you don't have to click through the whole book.

Overall I really enjoyed this project. It really got me involved in the material anc got me excited to learn new things, like css grid and visited links. I used these links in the gold underlined headers in this project.

Imitation

Another one of our projects was the imitation project. The purpose of this was to make a copy of an existing magazine or other printed media using only html and css.

In addition to the copying, we also had to make a version that can scale with the width of the webpage that it is opened on, this was also done with css.

This project was the first time we made a webpage that changes in real time. It is a very useful skillset that I was able to apply in almost every project I've done since, including this project.
